# MILES
A Max-Min Framework for Invariant Learning with Consistency and Diversity in Domain Generalization  (Pattern Recognition)


## Usage

1. Modify the file in the scripts
2. The main script file is `train.py` and `train_imagenet`, which can be runned by using `OfficeHome_VIT16.sh` from `scripts/OfficeHome_VIT16.sh`: `cd scripts; bash OfficeHome_VIT16.sh`.
```
cd /mlspace/MILES/script/
bash OfficeHome_VIT16.sh
```
